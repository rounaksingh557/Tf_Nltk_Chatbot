<h2>Setup Instruction: </h2>

> Setup Virtual Enviroment
>
> > For Windows

```sh
 python -m venv .
```

> > For Macos

```sh
pip install virtualenv
```

```sh
virtualenv .
```

> > For Linux(Debian)

```sh
sudo apt install virtualenv
```

```sh
virtualenv .
```

> Install Python Version

```sh
pip install -r requirements.txt
```
